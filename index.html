Projects

Online Blood Bank

Description: The Online Blood Bank project is a web application that connects blood donors with recipients. It provides a platform for users to register as donors, request blood, and manage blood donation events. The system also includes an admin panel for managing blood inventory and user data.

Technologies Used: HTML, CSS, JavaScript, PHP, MySQL

Features:

  • User registration and authentication
  • Blood request and donation management
  • Search functionality for finding donors and blood types
  • Admin panel for inventory and user management
  • Email notifications for blood requests and donations

My Role: Full-stack developer. I designed the front-end using HTML, CSS, and JavaScript. I developed the back-end using PHP and MySQL for database management. I also implemented the email notification system using PHP mailer.

Challenges and Solutions: One of the main challenges was ensuring the security of user data and blood inventory. I implemented secure authentication and authorization measures, including hashed passwords and user role management.
